## 声明文件模块化

一个类库如果很大，就会导致相应的声明文件也很大，因此引入了声明文件模块化的概念。

前面几节介绍了三斜线指令，它有两种方式引入文件。

```ts
//  寻找 node_modolues/@types/sizzle 文件夹下的声明文件
/// <reference types="sizzle" />

// 寻找相对路径下的某个·声明文件
/// <reference path="JQueryStatic.d.ts" />
```
